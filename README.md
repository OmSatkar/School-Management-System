# School Management System

This is a Java-console based School Management System project developed to manage student records, teacher details, fees, attendance, and other school activities.

The project provides a simple and user-friendly interface for performing basic school management operations.

## Features

- Student Management
- Teacher Management
- Attendance Management
- Fee Management
- Add / Update / Delete Records
- Search Functionality
- Database Connectivity

## Technologies Used

- Java
- JDBC
- MySQL
- Swing / AWT
- Eclipse IDE

## Modules

### Student Module
- Add Student
- Update Student
- Delete Student
- View Student Details

### Teacher Module
- Manage Teacher Information
- Assign Subjects

### Attendance Module
- Mark Attendance
- View Attendance Records

### Fee Module
- Manage Student Fees
- Payment Records

## Database

Database used: MySQL

Tables include:
- students
- teachers
- attendance
- fees

## Tools Used

- Eclipse IDE
- MySQL Workbench
- JDBC Driver
